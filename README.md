# RegiSmart

**RegiSmart** is an AI-driven platform designed to modernize university registrar transactions through an intelligent, interactive kiosk interface. Developed as part of **CPEPracDesign2**, this system combines advanced AI with multiple interactive features to provide a seamless, user-friendly experience for students and staff.

## Key Features

- **AI-Powered Automation** – Streamlines course registration, enrollment, and other administrative tasks.  
- **Voice Interaction** – Allows users to interact with the system using natural language commands.  
- **Touch Interface** – Intuitive touchscreen navigation for easy access and input.  
- **Camera Integration** – Supports facial recognition for identity verification and enhanced security.  
- **Authentication & Security** – Multi-factor authentication ensures secure access to sensitive academic data.  
- **Kiosk-Optimized Design** – Designed for self-service kiosks, reducing queues and administrative workload.
